# Housing-price-analysis
House pricing analysis using the R language and use of Ames data analysis
Housing Prices Examined Using R

A fresh look at home values begins with the Ames Housing data. From there, patterns start to show through careful review of each detail
The goal was to understand what factors influence house prices the most.

## Project Overview
- Cleaned and processed raw housing data
- Created new features such as total square footage and house age
- Used data visualization to explore trends and relationships
- Performed correlation analysis to identify key factors

## Key Findings
- Larger houses tend to have higher prices
- House quality has a strong impact on price
- Newer houses are generally more expensive
- Location plays an important role in pricing
Tools Used
- R
- dplyr
- ggplot2
- tidyverse
